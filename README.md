# CWL Demos

Demonstrations of CWL usage techniques and syntax

## Installation

```
make install
```

## Activate the environment

```
make bash
```

## Run a CWL

```
$ ./run-cwltool.sh custom-types/sampletypes.cwl custom-types/input.json
```
